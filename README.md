# DevPulse 🚀

A React learning project that integrates with the GitHub API to search developers, view profiles, browse repositories, and save favorite profiles.

This project was built to practice modern React concepts such as routing, component-based architecture, API integration, state management, and browser storage.

---

## 📖 About The Project

DevPulse allows users to:

- Search GitHub developers
- View public GitHub profiles
- Browse repositories
- Save favorite developers
- Persist favorites using Local Storage

The project focuses on learning React fundamentals while building a practical application with real-world API data.
